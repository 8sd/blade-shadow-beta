AppImage

One file to rule them all

The Deb file was designed to use Ubuntu libraries, which is very unconvinient for non-Ubuntu distribution as it forces to tweaks in a dirty way with the libraries and messing them up. Alex^#1629 did a really wonderful job by building this AppImage.

It embeds Ubuntu Bionic (18.04) libs and the official Shadow Beta application. However, it still needs a working VA-API and a compatible GLIBC. Moreover, a wrapper around the app itself will check your configuration and provides a diagnostic of the fatal errors.

The 2.0.0 version and greater uses the unified client. It supports the trackpad and the gamepads.

Work on: Do not work on:
Ubuntu 18.04 Ubuntu 16.04
Linux Mint 19 Linux Mint 18
Debian 10+ Debian 9
Arch Linux GalliumOS 2.2
Solus Budgie
Fedora 28
GalliumOS 3.0

To use it, just download the AppImage, put it where you want, make it executable and then launch it.

  • Download the AppImage

For lazy people, here is a set of command that do everything and put the AppImage in ~/.local/share/Shadow Beta/ folder: 

mkdir -p ~/.local/share/Shadow\ Beta
cd ~/.local/share/Shadow\ Beta/
rm -f shadowbeta-linux-x86_64.AppImage
wget --content-disposition 
chmod +x shadowbeta-linux-x86_64.AppImage
./shadowbeta-linux-x86_64.AppImage

Setup your GPU for the VA-API

As Shadow for Linux uses the VA-API, it is important to well setup your GPU for it.
So it is important to understand how works vainfo and how to debug it.

First things first, vainfo needs to be installed on your OS. It is a program that describes how your GPU is detected by the VA-API.

  • Debian based distribution: sudo apt install vainfo
  • Arch Linux: sudo pacman -S libva-utils
  • Solus: sudo eopkg it libva-utils

The vainfo when executed in a terminal will return the abilities of you GPU. There are 3 things to check:

  • If it returns nothing or returns -1 somewhere, it means that the GPU is not well detected. Your VA driver needs to be updated.
  • In the list of profiles, it mentions "H264" at least once. If supported, you can use the Shadow on Linux.
  • In the list of profiles, it mentions "H265" or "HEVC" at least once. If not, you will not be able to use H265 but as long as you have H264, you still can use Shadow on Linux.

Install the drivers

Even if Xorg recognizes your GPU, VA-API sometimes needs specific drivers to use it. The best way to install them is to follow the official documentation of your distribution about it:

  • Debian based
  • Arch Linux
You may also search in the repositories the name of your GPU with libva. For instance, on Solus with an Intel, sudo eopkg it libva-intel-driver.

Always check your vainfo. You do not need to reboot after the installation of the drivers.

For most cases

If your active GPU is an Intel chipset and you have no dedicated GPU, you will need the i965 Intel driver.

  • Ubuntu: sudo apt install i965-va-driver
  • Arch Linx: sudo pacman -S libva-intel-driver
  • Solus: sudo eopkg it libva-intel-driver

For NVIDIA users, you have 2 options:

  • Use the Nouveau driver and install the old NVIDIA firmware that provides support for the VA API.
  • Use the Arekinath patch to provides VA API compatibility for recent NVIDIA cards.

In both cases, we strongly recommend you to follow this link: https://gitlab.com/aar642/shadowos-boot#support

Shadow tool

If you still think you need to debug your application, a handful tool has been designed by the community to make it simple.
It lists all the error on your computer like a bad VA-API integration, missing libraries or miscellaneous stuff.

To use it, make sure that vainfo and python3 are installed on your computer. Then, to start it, follow the next command lines: 

git clone https://github.com/NicolasGuilloux/blade-shadow-beta.git
cd blade-shadow-beta/shadow-tools/
python3 main.py

If GTK is not available on your machine, you can print a report in the terminal using python3 report.py.

The update button refreshes the information shown in the window.
The pastebin button creates a detailed report and give you back the link to share it to the community.
The tabs gives extra information about your configuration: GPU, Libraries, Shadow Logs.
