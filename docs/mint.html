Linux Mint

We strongly recommand you to use the AppImage instead of following this tutorial as it makes things easier. Moreover, the wrapper helps to track error directly from the Shadow.

Shadow was designed natively for Ubuntu 18.04 (on which is based the latest version of Mint) and does not require a lot of tweaks. Still, some stuff need to be done before turning your Linux machine to a powerful gamer PC with Shadow.

First things first, download the deb file from the Shadow website by following this link https://macosx.update.blade-group.fr/mpl/linux/beta/bionic/shadowbeta.deb and install it.

The CURL library and the gstreamer for VA-API are often missing so just to be sure, run sudo apt install curl gstreamer1.0-vaapi.

Shadow does not work on Wayland, so if echo $XDG_SESSION_TYPE does not return x11, log out, click on the engine weel and select "Xorg" then log in again. It will look exactly the same but it will actually run on Xorg.

You will need also to add your user to the "input" group by doing: sudo gpasswd -a $USER input. You will need to reboot after this step.

For all distribution, you may experience an issue with the VA-API which is involved in the decoding process. Please follow the tutorial we made right here.
